最简单的脚本使用方法就是复制一份 (😂 [Ctrl] + [C] 大法)，而后清除所有的函数功能，只保留符合用户需求的那部分。

脚本里的变量 ispec_dir 应该指向 iSpec 的安装路径，比如：

#--- iSpec directory -------------------------------------------------------------
ispec_dir = '/home/virtual/iSpec/'
sys.path.insert(0, os.path.abspath(ispec_dir))
import ispec

在终端输入 python example.py 即可执行该脚本。
